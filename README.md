# Boggled, a Boggle Helper

## Dictionary API

This program uses the Merriam-Webster api available at https://dictionaryapi.com. It will
look for an environment variable called DICTIONARY_API_KEY. Make sure it is exported into
your terminal, like so:

```
export DICTIONARY_API_KEY=your-api-key-here
```

This program also assumes you're using the "collegiate" dictionary api.
